# Elastic Compute Cloud
The **main service** that allows AWS to be infrastructure as a service, it contains many services in one
- Renting Virtual Machines (EC2 Instances)
- Storing data on virtual drives (EBS Volumes)
- Distribute load across machines (Elastic Load Balancer)
- Scaling the services using an auto-scaling group (ASG)
